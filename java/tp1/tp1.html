TP 1 – Lecture écriture-Type des données-Priorité des opérations

Exercice 1 : Ecriture des données

  1. Affichez sur la console le message suivant : « Bienvenue au parcours Java ».
  2. Stockez votre nom dans une variable appelée « nom » puis en utilisant cette variable afficher le message suivant : « Bienvenue ‘nom’ au parcours Java » (exemple : si votre est Ali le message sera : « Bienvenue Ali au parcours Java »).
  3. Affichez sur la console de l’addition de deux valeurs numériques. (Exemple : si les valeurs sont 3 et 6 vous devez afficher « La somme vaut : 9 ». NB : la valeur doit être calculée dynamiquement).

Exercice 2 : Lecture des données

  1. Vous devez récupérer à partir du clavier le message suivant : « Heureux de débuter la formation en java ». Puis affichez-le sur la console.
  2. Reprendre la question 2 de l’exercice 1, avec la modification suivante : Vous devez vous-même saisir dynamiquement au clavier le nom auquel vous voulez souhaiter la bienvenue.
  3. Récupérez une valeur numérique à partir du clavier, puis multipliez la par 4 et afficher le résultat sous la forme suivante : « La valeur fois quatre vaut : résultat ».

Exercice 3 : Priorité des opérations

  1. Stockez dans une variable a le résultat de l’opération 3/4 et affichez-le.
  2. Soit l’opération 2+3*2-6/2, apportez les modifications nécessaires pour que le résultat final donne -4.0.
  3. Arranger l’opération 9+3*9%2 pour avoir comme 12 comme résultat.

Exercice 4 : Date

  1. Accédez et affiche en rouge sur la console la date courante de votre machine.
  2. Accédez et affichez sur la console l’heure courante de votre machine (12 :45 :55).
